\# AWS Secure Infrastructure Project



\## Overview



This project demonstrates a secure AWS architecture using:



\- VPC

\- Public and Private Subnets

\- Internet Gateway

\- Route Tables

\- Security Groups

\- EC2 Instances

\- OpenVPN

\- Amazon S3

\- Amazon CloudFront

\- AWS WAF

\- IAM Users and Roles



\## Architecture



Architecture diagram is available inside the Architecture folder.



\## Project Flow



\### User



\- AWS WAF

\- CloudFront

\- S3 Website



\### Administrator



\- OpenVPN

\- Private EC2



\## Security Features



\- VPN Based Access

\- Private Subnet Isolation
